About us

OUR MISSION

Our Mission is to develop a web application which recognizes the users gestures and performs the shortcut key operations like crtl + c, ctrl+v, alt+f4, etc. Users can use the application while he/she is involved in his daily works in the PC.The application runs in the background and user can also pause/resume the detection feature. Apart from default shortcut keys assigned in the application, users can also make their custom set of shortcut keys for the given gestures. Along with user credential login, face recognition feature is also integrated for better security and effective login.

Shortcut key Operations

GESTURES FUNCTIONALITY
Ctrl + A Selects all text, images, and other objects on the page
Ctrl + B Makes selected text bold
Ctrl+C Copy any selected text or another object
Ctrl + D In browser, creates a new bookmark/favorite for the current page. In Microsoft Excel and Google Sheets,it fills and overwrites a cell with the contents of the cell above it in a column. In Microsoft PowerPoint,it inserts a duplicate of the selected slide. In Microsoft Word,it opens the font preferences window. In the Linux command-line shell,logs out of the interface.
Ctrl + F In Microsoft Outlook, it forwards an e-mail. In all other major applications,it opens the find box that lets you search for characters, text, and phrases on the current page.
Ctrl + I Italicizes selected text
Ctrl + L In Microsoft Excel and all other spreadsheet programs,it opens the Create Table dialog box. In Microsoft PowerPoint, Microsoft Word and other word processors, the Ctrl+L keyboard shortcut aligns an object/selected text with the left side. When using the bash shell, you can clear the screen by pressing Ctrl+L. In all major Internet browsers pressing Ctrl+L brings the address bar (location bar/omnibox) into focus.
Ctrl + N In all major Internet browsers,it opens a new application window. In Microsoft Excel and most other spreadsheet programs,it creates a new blank worksheet. In Microsoft Outlook,it has multiple uses. For example, if you're in the Mail section, it creates a new e-mail message. If you're in the Calendar section, it creates a new appointment. In Microsoft PowerPoint, it opens a new blank slide deck in another PowerPoint program window. In Microsoft Word and other word processors,it creates a new document.
Ctrl + O Open a new document, page, URL, or other files.
Ctrl + P Open a print window to print the page you're currenlty viewing
Ctrl + S Save changes to a file
Ctrl + U Underlines selected texts
Ctrl + V Pastes any text, image, video, or other object into the active application
Ctrl + W Close/open tab in a browser or close a document in Word
Ctrl + X Cut a selected section of text, image, or other object.
Ctrl + Y In all major Internet browsers, Microsoft excel, Microsoft powerpoint pressing Ctrl+Y will redo any previously-undone action. In the edit command window,it deletes the current line.
Ctrl + Z Undo any change made in the currently opened file.
Ctrl + F5 Refreshes or reloads the page or document window
I Open the Miniplayer.
J Seek backward 10 seconds in player.
K Pause/Play in player.
L Seek forward 10 seconds in player.
F Activate full screen. If full screen mode is enabled, activate F again or press escape to exit full screen mode.
T Open the Player in Theater Mode.
Alt + F Open the file menu or file tab depending on the program you're using.
Alt +E Open the Edit file menu option for programs supporting a file menu
Windows key+Tab Cycle through open programs on the Taskbar.
Windows key+Ctrl+O Opens the Virtual Keyboard
Windows key+Down arrow Minimizes the screen. Also, when in the side by side viewing mode, this shortcut takes the screen back to a minimized size
Windows key+Up arrow When in the side by side viewing mode, this shortcut takes the screen back to full size.
Windows key+ I Open the Windows 10 Settings window.
Windows key+ D Display and hide the desktop.
Windows key+ L Lock device.
Windows key + Shift + S Capture part of the screen with Snip & Sketch.
Windows key+Left arrow Shrinks the window to 1/2 screen on the left side for side by side viewing.
Windows key+Right arrow Shrinks the window to 1/2 screen on the right side for side by side viewing.
Alt+Tab Pressing Alt+Tab moves from left-to-right in the available open windows.
Alt+f4 Used to close the currently-active window
End key The End key is a key found on a computer keyboard that moves the cursor to the end of the line, document, page, cell, or screen
Space key Gives space between words, letter,number etc., during typing
Enter key Return to the next line or starts a new paragraph in a word processor.Activate a button or another object.
Esc key Allows the user to abort, cancel, or close an operation.
Left Arrow Move the cursor left in the current line
Right Arrow Move the cursor right in the current line
Up Arrow Move the cursor Up.
Down Arrow Move the cursor Down.
F1 Activates help for current open application. If you're at the Windows desktop, this activates the help for Windows.
F2 Renames a highlighted icon, file, or folder in all versions of Windows.
F3 Starts find or search when at the Windows desktop in all versions of Windows.
F4 Opens the drive selection or address bar when browsing files in Windows Explorer.
F5 Allows you to refresh or reload the page or document window..
F6 While in Windows Explorer, F6 moves the cursor to different Windows Explorer pane or in Windows 7 browse available drives.
F7 Used to spell check and grammar check a document in Microsoft Apps.
F8 Pressing F8 over and over as the computer is booting starts the computer in Safe Mode.
F9 Refreshes a document in Microsoft Word and sends and receives emails in Outlook.
F10 Activates the file menu bar in all versions of Windows.
F11 Enters and exits full screen mode in Internet browsers.
F12 Opens the Save As dialog box in Microsoft Word
